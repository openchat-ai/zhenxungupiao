# 三进制与股票预测的玄学关系

> 理论扩充 · 震巽股票 · yoyo 平衡三进制引擎
>
> 本文阐述**为何**用三进制（而非二进制）做选股决策，以及它与《易经》象数传统、
> 股市波动哲学之间的对应关系。技术实现见 `ternary_signal.ty`；本文不谈吉凶断语，
> 只谈**符号体系与决策结构**的契合。

---

## 1. 震巽：变动的两种面相

「震巽股票」之名取自八卦中的**震**（☳，雷）与**巽**（☴，风）。

| 卦 | 象 | 股市隐喻 |
|----|-----|----------|
| **震** | 雷动、骤变、惊觉 | 放量突破、消息冲击、趋势启动 |
| **巽** | 风入、渗透、渐进 | 均线漂移、资金缓慢吸筹、指标钝化 |

二者同属**变动**，但一刚一柔：震是离散跳变，巽是连续渗透。K 线正是一根根「震」（OHLC 跳变）叠在「巽」（趋势底色）之上。

股票预测在玄学传统里从来不是「算出一个确定未来」，而是**察变动之机**——与震巽之道同构。

## 2. 为何是「三」，而非「二」？

### 2.1 阴阳不足，三态才完整

二元（涨/跌）只有对立，缺少**「不动」**这一态：

- 阴（跌）↔ 阳（涨）之间，必有**中和、蓄势、观望**。
- 《易经》每卦三爻，三爻才能表达「少阴阳」「多阴阳」与「纯阳」之间的全部过渡。
- 《易传》：「天三地二」「三才」——天、地、人；升、持、降。

映射到交易动作，恰好三种，不多不少：

| 平衡三进制 trit | 交易语义 | 玄学象 |
|-----------------|----------|--------|
| **+1** | 买入 | 阳、伸、进 |
| **0** | 持有 | 中和、养、待 |
| **-1** | 卖出 | 阴、敛、退 |

这是**平衡三进制**（balanced ternary）：`-1, 0, +1`，而非无符号 `0, 1, 2`。
yoyo 内部用 `0/1/2` 编码，是因为状态槽无符号；语义上仍是 `-1/0/+1`。

### 2.2 三进制与二进制的哲学差

| 体系 | 世界观 | 决策风格 |
|------|--------|----------|
| 二进制 | 非此即彼 | 满仓/空仓，易过拟合 |
| 三进制 | 含中间态 | 允许「看不清」，减少妄动 |

《道德经》：「道生一，一生二，二生三，三生万物。」
三，是**生成**的起点。股市里「持有」不是懒惰，而是承认**道在中**——信息未明时的正当姿态。

## 3. 集百家之长：多指标 → 唯一信号

### 3.1 四指标与四象

本 App 取四个技术指标，各压缩为一个 trit，再**投票求和**：

1. **均线**（快慢相对）—— 势
2. **趋势**（价相对均线）—— 位
3. **RSI** —— 气（过亢/过竭）
4. **MACD** —— 动量（阴阳消长）

四者对应**四象**之数（少阳、少阴、老阳、老阴的简化读法）：四个「专家」，各说一句「进 / 待 / 退」。

```
指标① 均线  → trit₁  ∈ {−1, 0, +1}
指标② 趋势  → trit₂
指标③ RSI   → trit₃
指标④ MACD  → trit₄
```

### 3.2 求和取符号：众议归一

决策规则（与 `ternary_signal.ty` 等价）：

```
S = trit₁ + trit₂ + trit₃ + trit₄     （范围 −4 .. +4）
```

| 条件 | 结果 | 语义 |
|------|------|------|
| S > 0 | **+1 买入** | 众阳胜 |
| S = 0 | **0 持有** | 阴阳平衡 |
| S < 0 | **−1 卖出** | 众阴胜 |

yoyo 实现里用无符号累加与中性值 **4** 比较（四票各 `0/1/2`，中性 sum=4），
与「signed sum 的符号」**数学等价**——四象归中，中数为 4。

这类似卦占中的**「综卦」「互卦」**：不迷信单一指标，而看**整体倾斜**。
一指标说谎，四指标同时说谎的概率更低——不是玄学 garantie，是**结构上的稳健**。

## 5. 三进制、yoyo 与「零依赖」的道

### 5.1 平衡三进制是最简完备决策代数

一个 trit 同时编码**方向**（正负）与**存在性**（是否偏离零）。
求和后的**符号**，就是《易》里「观卦定吉凶」的极简算术版——
不是算命，是**把「多象」压成「一断」**的形式化。

### 5.2 yoyo：三进制落地的语言

yoyo 用 9 个（及扩展）opcode、状态槽、handler——
**极简符号 → 可执行机器码**，与「易」之「简则不易」同旨：

- 不引外部库：象自具足，不借他力。
- `.ty` 自举编译：道在自身中完成循环（自指）。
- 多平台后端：同一「道」在不同「器」上显化（PE / ELF / 帧缓冲）。

## 6. 股票预测：我们能主张什么，不能主张什么

### 可以主张（结构与纪律）

- 三态动作空间符合交易伦理（不强迫二元满仓）。
- 多指标 trit 投票是一种**谦逊的集成**：承认单术不全。
- 震巽之名提醒：市场主象是**变**，策略应跟踪变，而非预测固定点。

### 不能主张（玄学边界）

- trit 求和**不能**保证盈利；过去指标不蕴含未来。
- 「紫买绿卖」是**视觉卦象**，非天命。
- 易经占筮重「心诚」「时机」；量化重「回测」「风控」——本系统只提供后者之**符号骨架**，占筮之义在人不在码。

## 9. 结语：术与象并行

震巽股票的设计，是**象数传统**与**极简计算**的一次合流：

- **象**：三态、四票、阴阳中和——对应人心对市场「进、待、退」的自然划分。
- **数**：平衡三进制求和、yoyo 零依赖编译——对应可审计、可自举的工程实现。

玄学不提供 α；它提供**命名的正当性**与**结构的节制**。
在「变动」之中，持有一个不为、少为、有为的三态开关——
这或许就是三进制与股票预测之间，最诚实的一种关系。
