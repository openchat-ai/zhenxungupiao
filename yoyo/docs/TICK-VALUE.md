# 逐笔 / L2 数据：真价值提炼（买之前必读）

> 震巽项目实证：`backtest_v4_*`、`backtest_v4_recent_tickcov_*`  
> 问题：**花几千~几万买 Wind/掘金/L2 包，会不会血亏？**

---

## 一、一句话结论

| 问题 | 答案 |
|------|------|
| 逐笔数据有没有价值？ | **有**，但在**微观结构 / 执行 / 短周期**，不在我们现在的用法 |
| 按当前「第 7 票」用法，买商用 L2 值不值？ | **大概率血亏**（万元买的数据，用成「一个日频数字」） |
| 免费 2026 八股逐笔够吗？ | **够验证**「这条路有没有 α」——已验证：**日线方向预测仍 ≈0** |
| 什么时候才值得买？ | 策略改成**日内/执行层**，且能算出**可量化节省或 IC>0** 之后 |

## 二、逐笔数据「真正值钱」在哪（业界共识）

逐笔 / L2 不是「更细的 K 线」，而是另一类游戏：

| 价值类型 | 时间尺度 | 典型用途 | 谁在用 |
|----------|----------|----------|--------|
| **执行优化** | 秒~分钟 | 减少滑点、拆单、选时机 | 私募实盘、算法交易 |
| **订单流不平衡** | 分钟~小时 | 短期价格压力、冲击成本 | 高频、做市 |
| **微观结构研究** | tick 级 | 买卖主动性、大单、队列 | 券商研究所 |
| **风控/验证** | 当日 | 「信号发出时盘口是否支持」 | 中频量化 |
| **事件确认** | 日内外 | 利好落地当天买盘是否真强 | 事件驱动 |

**不擅长的事**（也是我们现在在做的）：

> 把全天主动买%压成一个数 → 预测**次日涨跌** → corr ≈ 0

这在学术上和业界经验里都**不奇怪**：日频方向本来接近随机游走，逐笔里的日内信息到收盘已被吸收大半。

## 三、我们实际怎么用的（价值损耗链）

```
全天几万笔 tick
    ↓ 汇总成一个数
active_buy_pct（主动买笔数 %）
    ↓ 阈值 55/45
第 7 票 trit（买/持/卖）
    ↓ 与另外 6 票相加
state[22] 日频决策
    ↓ 用来赚
次日收益 corr ？
```

**损耗了 99% 的信息**：

- 丢掉：时段（开盘/尾盘）、大单、价格档位、成交簇、委托侧
- 丢掉：日内路径，只留日终比例
- 用错尺度：日频投票预测日频收益

等于用「显微镜采了一整天的血」却只量**平均体温**，然后问能不能**预测明天发烧**。

## 四、实证：免费数据已经证明了什么

### 全样本 v4（2018–2026，逐笔覆盖 ~5.7% 日）

| 指标 | 值 | 含义 |
|------|-----|------|
| corr(信号, 次日) | **≈ 0** | 方向预测力无 |
| v4 vs 动量 | 5/8 跑赢 | 相对收益，非预测准 |

### tickcov 窗（2026，逐笔 **100% 覆盖**，115 交易日）

| 指标 | 9 日窗（旧） | **115 日窗（扩样本后）** |
|------|-------------|-------------------------|
| corr | +0.12 | **−0.026** |
| v4 收益 | +0.5% | −8.4% |
| vs 动量 | 6/8 | **6/8** |

**提炼**：

1. **9 日的 corr 回升是噪声**——样本加长后打回原形。  
2. **逐笔全覆盖并没有带来方向预测力**。  
3. **第 7 票仍可能改善「相对收益」**（少亏、仓位节奏），但这是**风控/降噪**，不是「猜对明天」。  
4. 花 **1 万/年** 去买同样信息密度的日频汇总 → **不能**把 corr 从 0 变成 0.1，除非**改特征和预测目标**。

## 五、买 vs 不买：决策表

### 会血亏的买法 ❌

| 买法 | 为什么亏 |
|------|----------|
| 指望 corr 从 0 → 0.3 | 实证已否 |
| 全市场 L2 历史包 + 仍用第 7 票日频规则 | 信息浪费 + 存储/清洗成本 |
| Wind 全终端只为八股逐笔 | 终端 3 万+，大材小用 |
| 还没改策略先买 3 年数据 | 锁定沉没成本 |

### 可能不亏的买法 ✅（但要改玩法）

| 买法 | 前提 | 价值从哪来 |
|------|------|------------|
| **掘金专业版 ~1 万/年** | 改做**日内**回测（分钟/tick 目标） | 订单流 IC 在小时级可测 |
| **历史逐笔包（数千~数万）** | 只做**八股**、解压进 archive、**离线**用 | 一次买断，无 API 封 IP |
| **券商 L2 ~60 元/年** | 只用于**实盘看盘验证** | 人眼确认，不进回测 |
| **Wind/iFinD** | 同时要财报/宏观/全市场日线 | 逐笔只是副产品 |

### 震巽当前阶段建议

```
阶段 0（现在）：免费八股 2026 逐笔  →  已证明「日频第 7 票」不值万元
阶段 1（若要深挖）：用免费数据试「日内特征」→ 有 IC 再谈买
阶段 2（实盘前）：  券商 L2 或执行算法  →  为滑点付钱，不为 corr 付钱
```

## 六、若不买东西，怎么把「真价值」榨出来？

在**现有免费 2026 tick archive** 上，还可试（仍不进商用采购）：

| 特征 | 问的问题 | 和现在的区别 |
|------|----------|--------------|
| **尾盘 30 分钟主动买%** | 收盘前资金方向 → **次日开盘**跳空？ | 缩短预测 horizon |
| **上午 vs 下午 imbalance** | 日内结构是否预示午后？ | 用时段，不用全日 |
| **大单笔数占比** | 机构 vs 散户盘？ | 不只数笔数 |
| **作否决票** | 日线 6 票喊买，但 flow<45% → 强制持 | 降噪，不增预测 |
| **仅高振幅日用** | 平淡日不用第 7 票 | 与 `wuwen.ty` 一致风险联动 |

**验收标准**（买之前必须过）：

1. 样本外 **IC 或 corr** 在明确 horizon 上 **> 0.03 且稳定**（不是 9 日偶然）  
2. 或：实盘/仿真 **滑点下降 > 数据年费**（执行价值）

过不了 → **不买**。

## 七、和「一涨二跌三无问」哲学的关系

| 你的概念 | 逐笔能贡献什么 | 不能贡献什么 |
|----------|----------------|--------------|
| **众人一致买卖** | 当日主动买 55%+ 像「买盘一致」的**微观证据** | 证明不了「心电感应」 |
| **三无问** | flow 平淡 → 第 7 票持，多一路「无问」 | 不能替代 η 新闻层 |
| **三进制决策** | 第 7 票多一票结构 | 七票仍同源相关，≠ 七路独立情报 |

逐笔对震巽的**哲学价值** > **预测价值**：它让「一致买卖」从 K 线寓言变成**可观测盘口**；但哲学丰富 ≠ 钱包变鼓。

## 八、买商用数据前的检查清单

- [ ] 我要解决的是**执行滑点**还是**预测次日**？（后者已证伪）
- [ ] 特征是否**日频以上**？（是 → 先别买 L2）
- [ ] 免费 2026 八股上，新特征 **IC 回测**过了吗？
- [ ] 覆盖范围是否**只需 8 股**？（是 → 别买全市场包）
- [ ] 年费是否 **< 预期年化 alpha 或滑点节省**？

五项里若后三项都没做 → **买了血亏概率极高**。
