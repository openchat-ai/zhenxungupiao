# 震巽股票 · 实战级信号规范（SPEC-REALTRADE）

> 目的：把「达到实战级别」从一句口号变成**可证伪、可复现、分阶段验收**的工程规范。
> 配套证据：`make research-oos` → `research/oos/OOS_REPORT.md`（当前**无一信号达标**）。
> 本规范不承诺赚钱；它规定**在什么条件下**一个信号才**够格**投入实盘小资金。

---

## 〇、一句话

> 用 8 只蓝筹做「择时」几乎没戏（广度太小）；实战级的现实路径是
> **横截面选股（cross-sectional ranking）+ 严格防泄漏的 walk-forward 验证 +
> 扣成本/容量 + 多重检验/过拟合校正 + 分阶段实盘门槛**。
> 判据是 **ICIR / Deflated Sharpe / PBO**，不是单点 corr。

## 一、目标与非目标

| | 内容 |
|---|---|
| **目标** | 产出一个横截面信号组合，满足下方 Gate A/B/C，可支撑**实盘小资金**并有明确扩容/叫停规则 |
| **目标** | 全流程**点时数据（point-in-time）**、无前视、无幸存者偏差、复权正确 |
| **目标** | 成本、容量、涨跌停/停牌/T+1 约束**内生**进回测 |
| **非目标** | 预测指数级别的「集体狂飙/崩盘」（三进制理论已界定：不能预言集体） |
| **非目标** | 「保证收益」「稳赚」；任何不可证伪的表述 |
| **非目标** | 为过一次窗口而调参（数据窥探）——见 §5 |

## 二、为什么现状不达标（基线）

来自 `make research-oos`（2018–2026，8 股，扣成本，样本外）：

| 现状问题 | 证据 | 规范对策 |
|---------|------|---------|
| 单股择时、广度≈8 | OOS 主动信号夏普 ≤ 买入持有 | §3 扩到数百标的横截面 |
| 只做样本内 | v2–v6 全 IS | §5 walk-forward + purge/embargo |
| 忽略成本 | 七票毛 +17.9% → 净 +3.1% | §6 成本/容量模型 |
| 16 选 1 的窥探 | `flow_pure` t≈1.08 不显著 | §5 DSR + PBO + FDR |
| 单点 corr 作判据 | IC t 全 <0.6 | §5 ICIR/DSR 作判据 |

**关键数学（主动管理基本定律，Grinold 1989）**：

```
IR ≈ IC × √breadth
```

- 8 股择时：breadth 极小 → 即使 IC 真为正也放大不出 IR。
- 300 股 × 每年约 250 次横截面调仓：breadth 巨大 → **同样 0.02–0.03 的 IC 就可能形成可交易的 IR**。
- 这解释了为什么「corr≈0.003 无望」是针对**旧用法**；换成**宽横截面**后，小 IC 的价值被广度放大——这才是实战的现实入口。

## 三、数据规范（Data Spec）

| 项 | 要求 |
|----|------|
| **股票池** | 可复现的、点时的成分（如中证 300/500 历史成分），**含退市/调出**以除幸存者偏差 |
| **价格** | 前复权/后复权一致口径；对齐分红送转；剔除停牌日 |
| **点时性** | 任一特征在 t 日只能用 **≤t 收盘**可得的信息（财报按**实际披露日**而非报告期） |
| **基本面** | 点时财务（营收/利润/资产/现金流）、一致预期（如有） |
| **量价微观** | 日线 + 可选 L2/逐笔（见 `TICK-VALUE.md`：仅在日内/执行层才值得付费） |
| **另类** | 新闻/情绪 η（见 `perturbation.ty`）、资金流、龙虎榜等，均需点时化 |
| **QA 门** | 缺失率、极值、复权跳变、成分漂移的自动校验；不过 QA 门的日期不进回测 |

> 现有 `research/archive/hist_*.csv`（8 股）**仅够方法验证**，**不够**支撑实战广度。
> 第一步工程即扩池（见 §10 M1）。

## 四、因子/特征规范（每个因子先有经济学假设）

| 因子族 | 事前假设（可证伪） | 代表 | 本仓库现状 |
|--------|-------------------|------|-----------|
| 价值 | 便宜相对贵长期跑赢 | B/P、E/P、FCF/EV | 无（待建） |
| 质量 | 高盈利/低杠杆更稳 | ROE、毛利率、应计 | 无 |
| 动量 | 中期趋势延续 | 12-1 月动量 | 有（单股版） |
| 反转 | 短期过度反应修正 | 1 月反转、RSI | 有（单股版） |
| 低波/低beta | 低风险异象 | 波动率、beta | 无 |
| 流动性 | 非流动性溢价 | Amihud、换手 | 部分 |
| 微观结构 | 主动买卖/大单失衡 | flow、L2 失衡 | v5/v6 逐笔（样本小） |
| 情绪 η | 叙事同步情绪 | 新闻情绪分 | `news_eta` |

**处理规范**：横截面 **winsorize（如 1%/99%）→ 行业&市值中性化 → z-score**；
多因子合成用**稳健**方式（等权/IC 加权/正交化），**禁用**在全样本上拟合的权重。

## 五、验证协议（核心，防自欺）

1. **横截面 IC**：每期 Spearman(rank(factor_t), forward_return_{t→t+h})；
   汇总 **mean IC** 与 **ICIR = mean(IC)/std(IC)·√periods**。
2. **Walk-forward**：滚动/扩张训练窗 → **紧邻**样本外测试窗，只用历史决定当期。
3. **Purge + Embargo**（López de Prado）：因 forward return 有重叠，训练/测试间**清洗**重叠标签并留**禁运间隔**，杜绝泄漏。
4. **CPCV → PBO**：组合化清洗交叉验证，估 **Probability of Backtest Overfitting**，要求 **PBO < 5%**。
5. **Deflated Sharpe Ratio（DSR，Bailey & López de Prado 2014）**：按**试验次数 N**、偏度、峰度**去膨胀**夏普；要求在给定置信下 **DSR > 0**。同时记录并公开 **N（试过多少配置）**。
6. **多重检验 FDR**：因子族用 Benjamini-Hochberg 控制 FDR（或 Bonferroni 保守）；报告**有效独立检验数**。
7. **最短跟踪期 MinTRL**：给定目标夏普，算出**统计上可区分于 0**所需的最短样本长度，未达不予结论。
8. **稳健性**：牛/熊/震荡子样本 + 多 walk-forward 窗 + 多 `SPLIT`（现有 `research-oos` 已支持 `SPLIT` 敏感性）一致，非单窗侥幸。

## 六、成本、约束与容量（内生进回测）

| 项 | 建模 |
|----|------|
| 佣金 | 双边约万 2.5，含**最低 5 元/笔**下限 |

## 八、验收门槛（可复现的「实战级」定义）

> 全部为**样本外、扣成本、点时**口径。任一 Gate 不过即不得进入下一阶段。

### Gate A — 研究通过（离线）
- 横截面 **ICIR ≥ 0.5**（年化），且 mean IC 的 t 经 **FDR/Bonferroni 校正后显著**；
- **DSR > 0**（在 N 次试验、真实偏度峰度下，置信 ≥95%）；
- **PBO < 5%**；
- **净夏普**在 **≥2 个独立 walk-forward 窗** 且 **≥2 种市场状态** 下 **稳定为正**，并**风险调整后跑赢买入持有 / 基准**；
- 容量估计 ≥ 目标资金；换手在预算内。

### Gate B — 纸面前瞻（paper / 前视样本外）
- 冻结参数，前瞻若干**独立再平衡期**做纸面交易；
- 实盘化滑点/成交 **≈ 模型**（偏差在容差内）；
- 实盘 vs 回测 **tracking error 在容差内**，无隐藏泄漏迹象。

### Gate C — 实盘小资金
- 小资金上线，**硬性**回撤/换手/单名限额 + 熔断 + 监控；
- 现实收益累计达 **MinTRL** 且指标未失效 → 方可**逐步扩容**；
- 任一叫停判据触发（§11）→ 立即降杠杆/下线。

## 十一、失败判据 / 何时叫停（诚实边界）

- Gate A 反复不过、或过则依赖单窗/单因子 → **判定此路线在本数据上无稳定 α**，停止，不投实盘。
- 实盘 tracking error 持续超容差、或滑点显著劣于模型 → 下线复盘。
- 因子拥挤度飙升 / IC 结构性衰减 → 降杠杆或退役该因子。
- 触及硬性 max DD → 熔断。

> **底线**：任何阶段都不得为了「看起来达标」而放宽 §5 的防泄漏与去膨胀。
> 宁可诚实地得出「没有 α」，也不把窥探结果包装成实战级。
