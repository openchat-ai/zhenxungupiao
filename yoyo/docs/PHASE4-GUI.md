# Phase 4：GUI 帧缓冲

## 当前实现：`lib/chart.ty`

纯 `.ty` 软件渲染，像素写入固定 state 槽：

| 区域 | 槽位 | 内容 |
|------|------|------|
| K 线 | `0x80..0x89` | 涨红 / 跌绿 |
| 四指标 | `0x8b..0x8e` | 紫买 / 绿卖 / 灰持 |
| 综合信号 | `0x8f` | 紫买 / 绿卖 |

## 编译器 Opcode（0xB0–0xB4）

`make bootstrap` 合并补丁后可用动态帧缓冲。

## 配色

- **买入** `#a855f7` → `0xF7`
- **卖出** `#2ebd85` → `0x85`
